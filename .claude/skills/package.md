# 納品パッケージ Skill

完成したLPを納品用にパッケージ化する。

## 入力

ユーザーからクライアントフォルダのパスが提供される:
- 例: `clients/2026-03-15_bloom-salon/`

## 手順

### Step 1: 納品物の確認

1. `dist/` フォルダの内容を確認する
2. 以下のチェックを行う:
   - `index.html` が存在するか
   - プレースホルダー（`{{`）が残っていないか
   - 壊れたリンクがないか
   - 画像URLが有効か

### Step 2: README 生成

`dist/README.md` を自動生成する。内容:

```markdown
# {ビジネス名} ランディングページ

## ファイル構成
- `index.html` — メインページ

## サーバーへの設置方法
1. このフォルダの内容をWebサーバーにアップロードしてください
2. `index.html` がトップページとして表示されます

## カスタマイズ方法

### テキストの変更
`index.html` をテキストエディタで開き、変更したいテキストを直接書き換えてください。

### カラーの変更
`index.html` の `<style>` タグ内にある以下のCSS変数を変更してください:
- `--color-primary`: メインカラー
- `--color-secondary`: サブカラー

### 画像の差替え
画像URLを変更したい場合は、`src="..."` の部分を新しい画像のURLに書き換えてください。

## サポート
ご不明点がございましたらお気軽にお問い合わせください。
```

### Step 3: ZIP作成

```bash
cd clients/{client_folder}/dist
zip -r ../delivery.zip .
```

### Step 4: 納品メッセージ生成

クライアントへの納品メッセージを生成する:

```
お世話になっております。
LP制作が完了いたしましたので、納品いたします。

【納品物】
・index.html（メインページ）
・README.md（設置手順・カスタマイズ方法）

【確認方法】
ZIPを解凍し、index.html をブラウザで開いてご確認ください。
スマートフォンでの表示もご確認いただけますと幸いです。

修正がございましたらお気軽にお申し付けください。
（{修正ポリシーに基づく残り回数}回まで無料で対応いたします）

よろしくお願いいたします。
```

## 出力

1. `dist/README.md` ファイル
2. `delivery.zip` ファイル
3. 納品メッセージのテキスト（コピペ用）
